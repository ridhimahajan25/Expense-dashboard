# 💸 Expense Dashboard

A zero-friction monthly expense tracker built with React and Recharts.
Features AI auto-categorisation, budget tracking, streak rewards, and responsive charts.
No sign-up required data stays local.

## Live Demo
[View Live →]([https://expense-dashboard.vercel.app](https://expense-dashboard-lvklqvd6h-ridhimahajan25s-projects.vercel.app/)

## Features
- AI auto-categorisation from natural language notes
- Monthly budget tracking with progress bar
- Category breakdown pie chart and 6-month trend bar chart
- Streak counter for consecutive months under budget
- Fully responsive: works on mobile and desktop
- Zero backend, zero sign-up, data stored locally

## Stack
React · Recharts · Vite · Claude AI
